# jQuery RickRoll plugin v3.1.88.5rc12a

## What is it?
Enhance the UX of any website or app by randomly RickRolling your users!

BONUS: A bonus feature for those special users who roll with IE.

## Usage
```js
$("a:visible").rickroll();
```
